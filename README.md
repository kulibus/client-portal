# Client Portal

Web application developed as part of a web development training program.

## Description
This project is a simple client portal built with PHP, HTML, CSS and basic JavaScript.  
It allows handling user data through forms and server-side processing.

## Technologies
- PHP
- HTML5
- CSS3
- JavaScript
- Apache

## Features
- Client data input through forms
- Server-side data processing with PHP
- Basic frontend interface
- Project structure following good practices

## Purpose
Educational project developed from scratch to practice full web development fundamentals.
